# Jellyfin Video Tuner

A Javascript injection for Jellyfin that adds a "Video Tuner" menu to the player. It includes **Spatial Sharpening** (Convolution Matrix), Denoising, and standard picture controls (Brightness, Contrast, Saturation, Hue).

### Credits & Tech

* **Spatial Sharpening Engine:** Based on the convolution matrix logic from [IMFX Spatial](https://mebble.github.io/imfx/spatial.html).
* **Conflict Killer:** Automatically neutralizes CSS interference from other browser extensions.
